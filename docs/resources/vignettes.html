Case studies

Over time we have developed a number of examples, or case studies to help users experience how to use the R packages we have developed to perform a number of common tasks. Feel free to use the code provided as templates forming the starting point for your own analysis.

Buttons (e.g. Line transect) can be clicked to filter the examples. Reload the page to remove filter.

Analysing indirect surveys

multipliers
line transect
analysis
Estimation of Sika deer density based upon deer pellet distance sampling data. Includes estimation of decay rate using logistic regression and multiplier based on dung production rates. Using data from Marques et al. (2001).

Analysis of camera trap data using distance sampling

camera traps
point transects
analysis
Replicates analyses of duiker data presented by Howe et al. (2017) including model selection methods in presence of overdispersion (Howe et al. 2019)

Analysis of cue count surveys

cue counts
line transect
analysis
Cue counting example based on line transect data of winter wrens from Buckland (2006). Incorporating song rate in analysis.

Analysis of double observer data

mark-recapture distance sampling
line transect
analysis
Full independence and point independence analysis of golf tee data from multiple observers.

Analysis of line transect data

line transect
songbirds
analysis
Line transect data of winter wrens in Scotland. An example of model fitting, model selection, goodness of fit assessment. Basic analysis skills required for any distance sampling analysis.

Analysis of point transect data

point transects
songbirds
analysis
Point transect analysis of savannah sparrow data set from Knopf et al. (1988)

Analysis of stratified surveys

stratification
point transects
analysis
Analysis of stratified survey design, including model selection to determine whether stratum-specific detection functions are necessary. Uses point transect analysis of savannah sparrow data set from Knopf et al. (1988).

Assessing analysis decision effects via simulation

truncation
line transects
simulation
Simulate distance sampling surveys examining the effect of truncation distance and pooling robustness upon estimates of animal density (when true density is known).

Combating the effects of size bias

group size
analysis
Because of the detection process, average group size in sample is a positively biased estimate of average group size in population. This bias can be compensated by employing group size as a covariate in the detection function.

Getting started with dssd

line transect
design
dssd
Create a survey design within a study-area, and examine properties of the design such as how well it covers the study area and average transect lengths. The example uses a parallel line transects; other designs available in the dssd package follow from this example.
